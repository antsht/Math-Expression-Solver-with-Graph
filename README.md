# Math-Expression-Solver-with-Graph
## Program in C which parse an expression with one argument x and draw its graph in terminal


List of operations:

- `+` = +
- `-` = - binary
- `~` = - unary  
- `*` = *
- `/` = /
- `(` = (
- `)` = )
- `s` = sin(x)  
- `c` = cos(x)
- `t` = tan(x)
- `g` = ctg(x)
- `q` = sqrt(x)
- `l` = ln(x)
- `v` = if value is stored
